# AnimeHako

Anime discovery and management web application based on YummyAnime API.

## Features

### API realization

- **Authentication:** registration, login
- **Anime browsing:** catalog with search, filters (genres, year, rating, tags), sorting
- **Anime details:** full info
- **User management:** profile, anime list (watching, planned, completed, dropped)
- **Favorites:** add/remove anime from favorites

### Extra features

- **Anime Mathcer**: Tinder-style swipe interface for anime discovery.
- **Anime Tournament**: Bracket-style anime titles tournament system.
